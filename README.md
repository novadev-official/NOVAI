NOVAI is a premium CLI tool designed for security enthusiasts and Linux power users. It provides a sleek, modern interface for common tasks and a mock AI assistant.

## Features
- **Sleek ASCII UI**: Built with `rich` for a professional, high-end terminal experience.
- **Network Scanner**: Quickly scan for open ports on target IPs.
- **AI Assistant (Mock)**: A terminal-based assistant to help with security concepts.
- **System Dashboard**: Real-time view of your system's core specifications.

## Installation

### Linux / Kali Linux
1. Clone the repository or download the files.
2. Run the installation script:
   ```bash
   chmod +x install.sh
   sudo ./install.sh
   ```
3. Launch by typing:
   ```bash
   novai
   ```

### Windows
1. Install Python (if not already installed).
2. Install dependencies:
   ```cmd
   pip install -r requirements.txt
   ```
3. Run the tool:
   ```cmd
   novai.bat
   ```

## Requirements
- Python 3.x
- `rich` library
